## 关于码书

“码书”是一个开源的GitHub组织，我们的目标是创作360本技术迷你书，帮助程序员（Coder）用最简单、最易懂的方式学习和掌握各类技术知识。

无论是编程入门、框架使用、工具技巧还是底层原理，我们都希望通过“迷你书”的形式，让复杂的技术变得清晰直白——每本书聚焦一个具体知识点，篇幅精炼，阅读门槛低，让你能利用碎片时间快速吸收。
